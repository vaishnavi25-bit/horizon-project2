#PROJECT : MOVIE RECOMMENDATION SYSTEM USING BASIC FILTERING

## Overview
This project is a simple Movie Recommendation System built using Python and Pandas. It recommends movies based on user preferences by applying basic filtering techniques on a movie dataset.

## Objective
To recommend movies that match the user's selected genre, minimum rating, and release year.

## Features
* Genre-based movie filtering
* Rating-based movie filtering
* Year-based movie filtering
* Displays top matching movie recommendations
* Handles cases where no matching movies are found

## Technologies Used
* Python
* Pandas
* CSV Dataset

## How It Works
1. Loads the movie dataset from a CSV file.
2. Accepts user inputs for genre, minimum rating, and release year.
3. Filters movies based on the selected criteria.
4. Sorts the results by rating in descending order.
5. Displays the recommended movies.

## Conclusion
This project demonstrates the use of basic filtering techniques in Python to build a simple movie recommendation system. It helps users discover movies that match their preferences based on genre, rating, and release year.
